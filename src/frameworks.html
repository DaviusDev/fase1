¿Qué son los Frameworks?


Para poder empezar a indagar sobre frameworks es necesario entender que es realmente un framework, básicamente un framework es una serie de herramientas, estilos y librerías mayormente gratuitas que se usan para el desarrollo.

En este caso para desarrollo web, existen miles de frameworks disponibles en el mercado, que se caracterizan principalmente por su completitud, es decir viene con las librerías necesarias para que el proceso de desarrollo sea mucho más práctico y ordenado, lo que reduce considerablemente los tiempos de desarrollo.

Con esta aclaración hecha, pasemos a conocer algunos de los frameworks web que existen en el mercado, sus ventajas y sus desventajas.

  1. React:

    2. React: sin duda alguna React es el framework más conocido del desarrollo web fue desarrollado por Facebook junto a una comunidad de desarrolladores independientes, fue lanzada en mayo de 2013 y desde entonces ha sido mantenida por una comunidad activa, a continuación, sus ventajas y desventajas:

    Ventajas:

    • Es muy rápido y eficiente.
    • Facilita la reutilización de código
    • Utiliza Virtual DOM para actualizaciones efectivas
    • Es muy popular por ende tiene una Amplia comunidad de desarrolladores.

    Desventajas:

    • Su curva de aprendizaje es muy alta.
    • Depende de algunas bibliotecas externas.

    Enlace:

    https://es.react.dev/

  2. Angular:

    3. Angular es otro framework bastante conocido en el desarrollo web, desarrollado y mantenido por Google Angular actualmente cuenta con una base de usuarios bastante grande a nivel mundial su lanzamiento fue en septiembre de 2016.

    Ventajas:

    • Ofrece una solución completa sin depender de terceros.
    • Al ser mantenido por Google su desarrollo y soporte es constante.
    • Usa TypeScript lo que mejora la detección de errores.

    Desventajas:

    • Debido a su tamaño, su curva de aprendizaje es muy alta.
    • Las aplicaciones tienden a ser de un tamaño mayor comparado con otros frameworks.

    Enlace:

    https://angular.dev/

  3. Vue.js:

    4. es un framework de desarrollo enfocado en la construcción de interfaces de usuario su popularidad no es tan alta como la de los frameworks anteriormente estudiados, su lanzamiento fue en febrero de 2014.

    Ventajas:

    • Una curva de aprendizaje menor en comparación con otros frameworks.
    • Al ser MVC permite una separación entre componentes y directivas.
    • Su tamaño es mucho más reducido.

    Desventajas:

    • Su comunidad es mucho menor.
    • Al ser desarrollado en china,gran parte de su documentación se encuentra en chino, lo que dificulta consultarla.

    Enlace:

    https://vuejs.org/

Por otra parte, para despliegue de servicios web, existen más frameworks que funcionan con lenguajes específicos, entre ellos tenemos a:

  • Django, lenguaje Python
  • Express: Framework de Node.js lenguaje Js
  • Flask: Framework de Python
  • Ruby on Rails: Framework de Ruby lenguaje Ruby
  • Spring: Framework de Java lenguaje Java

Como Podemos observar existen infinidad de frameworks para cada uno de los lenguajes más populares en el mercado, lo importante es buscar el que más se adapte a nuestro proyecto para poder utilizarlo.
